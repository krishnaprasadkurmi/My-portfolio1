# Personal Portfolio Website

## Project Overview

This project is a Personal Portfolio Website developed using HTML5. The website showcases personal information, technical skills, and contact details.

## Objectives

* Create a professional portfolio webpage.
* Practice HTML5 structure and semantic tags.
* Organize content into multiple sections.
* Demonstrate understanding of web development fundamentals.

## Features

* Responsive layout
* About Section
* Skills Section
* Contact Section
* Navigation Menu
* Semantic HTML Elements

## HTML Concepts Learned

### HTML5 Structure

HTML5 provides a standard structure for building web pages using elements such as:

* html
* head
* body
* title
* meta

### Semantic Tags Used

* header
* nav
* main
* section
* footer

These tags improve readability, accessibility, and SEO.
